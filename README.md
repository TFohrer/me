## Deployment 

Site ist automatically build and deployed using github Actions.

## Development notes 

Project setup with `parcel`

### To run in development mode
``` 
yarn dev
 
# or

npm run dev
```
 
### To run in production mode
```
yarn build
# or
npm run build 
```
